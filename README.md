# Plane Battle Game

This repository contains a simple plane shooter game implemented with **pygame**.

## Requirements
- Python 3.12+
- pygame (install with `pip install pygame`)

## How to Run
1. Install dependencies:
   ```bash
   pip install pygame
   ```
2. Run the game:
   ```bash
   python3 plane_game.py
   ```

Use the **Left** and **Right** arrow keys to move the plane. Press **Space** to shoot. Avoid enemies and try to survive as long as possible!
